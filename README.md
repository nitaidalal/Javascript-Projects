## 📦 How to Use

1. **Clone the repository**

```bash
git clone https://github.com/nitaidalal/javascript-projects.git
```
2. **Open any project**
```bash
cd javascript-projects/ProjectName
```
3. **Run the project**

- Open the index.html file directly in your browser.
